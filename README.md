# Dueterium UI 🧬

**Dueterium** is a minimal yet punchy Svelte component library handcrafted for fast development, clean layouts, and that sweet sweet NeoUI styling vibe.

> Built with frustration, fixed with flair.

---

## 🚀 Features

- 🎨 Customizable `Button` and `Card` components
- ✨ Variant-based styling (`primary`, `success`, `warning`, etc.)
- 📦 Typed props and hover transitions
- 🔧 Ready to plug into Vite, SvelteKit, or vanilla Svelte setups
- 🧪 Designed for modular reuse and theme extension

---

## 📦 Installation

```bash
npm install dueterium-ui
```
